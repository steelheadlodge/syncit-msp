SyncIT — Apple Mail signatures

Built for Apple Mail’s quirks: smaller logo, blue divider as a real table cell (not CSS border), so Mail is less likely to stack the mark or turn the rule into a quote bar.

In Mail → Signatures: uncheck “Always match my default message font”. Select only the signature table → Copy → paste into the signature pane. Gmail/Outlook: use email-signature.html instead.
